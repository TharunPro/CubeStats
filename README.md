## CubeStats ⚡
A personal speedcubing analytics dashboard built with Python and Streamlit.
Upload your csTimer solve history and instantly get detailed stats, charts and insights about your progression.

## Features
- 📊 **Overview** — Personal best, Ao5, Ao12 and total solve count
- 📈 **Progression** — Visual chart of your solve times over time with rolling averages and expected competition average
- 🔍 **Insights** — Solve time distribution, consistency score, best time of day to cube and session stats

## How to use
1. Export your solves from csTimer (Session menu → Export → CSV)
2. Upload the CSV file on the home page
3. Navigate through the pages using the sidebar

## Running locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech stack
- Python
- Streamlit
- Pandas
