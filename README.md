# CubeStats 🎲

A personal speedcubing analytics dashboard built with Python and Streamlit.
Upload your csTimer solve history and get detailed stats and insights about your progression.

## Features

- 📊 **Overview** — Personal best, Ao5, Ao12 and total solve count
- 📈 **Progression** — Visual chart of your solve times over time with rolling averages
- 🔍 **Insights** — Solve time distribution, session stats and consistency score

## How to use

1. Export your solves from csTimer (Session menu → Export → CSV)
2. Upload the CSV file on the home page
3. Navigate through the pages using the sidebar

## Running locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech stack

- Python
- Streamlit
- Pandas
- Matplotlib
